# JARVIS
An intelligent AI-powered virtual assistant built to automate tasks, answer queries, and provide a smooth interactive experience through voice and text commands. This project demonstrates the integration of AI, automation, and voice recognition into a simple desktop assistant. 

**Features**
* Voice command support
* Open YouTube, WhatsApp, and websites
* Make calls and perform basic automation tasks
* Launch desktop applications
* Smart assistant interaction
* Easy-to-use and lightweight system
